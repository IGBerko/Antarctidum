## ✨ Features

### 🖥️ Editor
- **Monaco Editor** — full IntelliSense, syntax highlighting for 50+ languages
- **Catppuccin Mocha** theme — beautiful dark theme, easy on the eyes
- **Bracket pair colorization** — nested brackets in different colors
- **Minimap** — code overview on the right side
- **Multi-cursor editing** — `Ctrl+D`, `Alt+Click`
- **Find & Replace** — with regex, case-sensitive, whole word support
- **Auto save** — configurable delay
- **Format on save** — automatic code formatting
- **Customizable** — font size, font family, tab size, cursor style, and more

### 📁 File Explorer
- **Tree view** — expandable folder structure
- **File operations** — create, rename, delete files and folders
- **Context menu** — right-click for quick actions
- **File icons** — emoji-based icons for 30+ file types
- **Exclude patterns** — hide `node_modules`, `.git`, etc.

### 💻 Terminal
- **Integrated terminal** — powered by xterm.js + node-pty
- **Full PTY support** — real shell (PowerShell, bash, zsh)
- **ANSI color support** — 256 colors, true color
- **Catppuccin theme** — matching terminal colors
- **Resizable** — drag to resize terminal panel
- **Fallback mode** — simple exec terminal if node-pty unavailable

### 🔍 Search
- **Search across files** — find text in entire project
- **Regex support** — search with regular expressions
- **Case sensitive** — toggle case matching
- **Whole word** — match complete words only
- **Replace all** — bulk replace in current file
- **Click to navigate** — jump to search result

### 🧩 Plugin System
- **Marketplace** — browse and install plugins from server
- **Plugin API** — editor, UI, settings, file system access
- **Status bar items** — plugins can add status bar widgets
- **Hot reload** — reload plugins without restarting
- **Developer portal** — upload your own plugins at [antarctidum.itrypro.ru](https://antarctidum.itrypro.ru)

### 🌍 Localization
- **English** — full interface translation
- **Русский** — полный перевод интерфейса
- **Easy to extend** — add new languages in `app.js`

### ⚡ Other
- **Command Palette** — `Ctrl+Shift+P` — quick access to all commands
- **Custom titlebar** — frameless window with native controls
- **Keyboard shortcuts** — VS Code-like keybindings
- **Zoom** — `Ctrl+=` / `Ctrl+-`
- **Tabs** — multiple files, close/close others/close all
- **Breadcrumb** — file path navigation
- **Notifications** — toast messages for actions
- **Settings UI** — visual settings editor with search
- **Sidebar resize** — drag to resize sidebar
